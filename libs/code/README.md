# code - 通用代码库

收集的 TypeScript/JavaScript 通用代码片段。

## 模块

| 模块 | 说明 |
|------|------|
| md5 | MD5 加密工具函数 |
| string | 字符串处理工具函数 |
| ecs | 简易实体组件系统 (ESC) |
| dictionary | 字典数据结构实现 |

## 使用

直接引入需要的模块：

```typescript
import { md5 } from './src/md5/md5Util';
import { formatString } from './src/string/stringUtils';
```
